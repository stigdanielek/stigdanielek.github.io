Boende

Önskas övernattning rekommenderar brudparet boende på Strandhotellet i Öregrund, beläget 23 kilometer från Forsmarks bruk. Buss kommer att finnas tillgänglig både för transport till vigseln och tillbaka efter festen.

På Strandhotellet har brudparet preliminärbokat ett antal rum för er. För att boka dessa rum kontakta Strandhotellet på telefonnummer {{ site.data.general.hotelPhone }} så snart som möjligt, dock senast den {{ site.data.general.hotelDeadline }}. Uppge brudparets namn vid bokning.
